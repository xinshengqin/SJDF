
These Python scripts can be used to download topo if you have netcdf
installed:

fetch_etopo1.py
    fetchs 1 arcminute etopo1 data
coarsen_PS_PT_SJdF.py
    coarsens Puget Sound, Port Townsend and 
    the Strait of Juan de Fuca grids to 2 arcseconds
fetch_DiscoBay.py   
    fetchs 1/3 arcsecond DEM around Discovery Bay
